req_02_format

q_1a__ name_list       - Codename convention requirements
q_1a_1    format       - Required codenames for formatting implementation:
                          1) format - Core formatting utilities for content display
                          2) circle - Interactive circular container element
                          3) circle_shadow - Dynamic shadow beneath circle element

q_2a__ format          - Formatting requirements
q_2a_1    background   - Style page background:
                          1) Set page background to a neutral gray color
                          2) Ensure background extends to all edges of viewport
                          3) Maintain consistent background across different screen sizes
q_2a_2    text         - Style text elements:
                          1) Style "hello" text:
                             - Apply bright red color for visibility
                             - Use system default sans-serif font
                          2) Style "world" text:
                             - Apply bright blue color for contrast
                             - Match font family with "hello" text
                          3) Ensure both words maintain proper spacing and alignment
q_2a_3    circle       - Add circular container:
                          1) Create circular container:
                             - Draw perfect circle around combined text
                             - Maintain consistent padding around text
                             - Use contrasting background color
                          2) Implement shadow effect:
                             - Create shadow at 80% of circle size
                             - Position shadow to create floating effect
                             - Apply soft blur for natural appearance
                             - Ensure shadow remains distinct from background
                          3) Add hover interaction:
                             - Move shadow downward on hover
                             - Increase shadow size slightly during hover
                             - Create smooth transition for all movements
                             - Maintain visual hierarchy during interaction
