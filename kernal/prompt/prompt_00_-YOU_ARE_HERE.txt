prompt_00_-YOU_ARE_HERE

This file serves as a dynamic marker indicator that shows which prompt file 
currently contains the /// YOU ARE HERE /// marker.

This file will be renamed as the marker moves from one prompt file to the next,
using the numbering scheme to position it alphabetically above the active prompt file.

The hyphen (-) in the file name ensures this file always sorts before the 
corresponding prompt file in directory listings.

Current active prompt: prompt_01_start.txt (current)
Previous prompt: None (reset position)

When "reset" is executed, this file will reset to prompt_00_-YOU_ARE_HERE.txt
